=====================
FAQs: CDAP and Clouds
=====================

.. .. rubric:: Configuration: General


Can CDAP be run in a cloud ?	
-----------------------------
Yes. CDAP can run on either public or private clouds.

Which public clouds has CDAP been tested on?
--------------------------------------------
CDAP has been tested on these clouds: AWS (Amazon), GCE (Global Computer
Enterprises), Rackspace, Joyent, and DigitalOcean.

Do you have customers who run CDAP in production on a public cloud?	
--------------------------------------------------------------------
Yes, Cask has customers who run in production on public clouds. Currently, they are using
the Google Cloud Platform (GCP) and Amazon Web Services (AWS), but soon there will be
support for Microsoft's Azure. 

.. How can I install CDAP on EMR ?	
.. --------------------------------

.. 
.. How can I install CDAP on Azure?	
.. ---------------------------------
.. CDAP is available in the Microsoft Azure Marketplace. You can use the Azure interface to
.. easily install CDAP on Azure.
